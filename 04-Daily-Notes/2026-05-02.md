# Daily Notes - 2026-05-02

## Fokus Hari Ini

Mempelajari dan mempraktikkan kerentanan **Local File Inclusion (LFI)** melalui mekanisme *Directory Traversal* pada server lokal.

## Progres & Temuan

1. **Perbaikan Pathing:** Mempelajari cara menggunakan `os.path.abspath(__file__)` untuk memastikan server Flask menemukan folder target secara konsisten di Windows.
2. **Bypass Folder Limits:** Berhasil menggunakan payload `../` untuk keluar dari folder `documents` dan membaca file internal aplikasi (`app.py`).
3. **System Access:** Berhasil membaca file sistem Windows `C:\Windows\System32\drivers\etc\hosts` melalui browser menggunakan teknik traversal yang dalam.

## Pelajaran Penting

- Memahami bahwa `os.path.join` tidak aman jika salah satu argumennya adalah input mentah dari user yang mengandung path navigasi.
- Menyadari bahaya *Source Code Disclosure*; dengan membaca `app.py`, penyerang bisa memahami seluruh struktur keamanan aplikasi.
