# Daily Notes - 2026-04-21

---

## Project / File
Lab DOM XSS sederhana:

- Reflected DOM XSS via URL parameter `name`
- Stored DOM XSS via input status + localStorage

---

## Materi yang Dipelajari

### 1. Reflected DOM XSS

Mengambil input dari URL:

```javascript
const urlParams = new URLSearchParams(window.location.search);
let userName = urlParams.get('name');
````

Lalu dirender menggunakan:

```javascript
document.getElementById('greeting').innerHTML = "Halo, " + userName;
```

Payload uji:

```text
?name=<img src=x onerror=alert(1)>
```

### Insight

Jika data user masuk ke `innerHTML`, script dapat dieksekusi.

---

### 2. Stored DOM XSS

Input user disimpan ke:

```javascript
localStorage.setItem('user_status', input);
```

Lalu saat halaman direfresh:

```javascript
display.innerHTML = "Status Tersimpan: " + savedStatus;
```

Payload uji:

```html
<img src=x onerror=alert('stored-xss')>
```

### Insight

Payload tetap tersimpan dan aktif saat page reload.

---

## Sink Berbahaya Hari Ini

* `innerHTML`

---

## Cara Fix Aman

Gunakan:

```javascript
textContent
```

Contoh:

```javascript
display.textContent = input;
```

## Pelajaran Penting

3. `innerHTML` sangat berisiko jika menerima input user.
4. Membuat lab sendiri membuat pemahaman jauh lebih cepat.
